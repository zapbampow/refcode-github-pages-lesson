Github Pages

Let others see your website by deploying it using Github pages.

Steps

  1. Create a repository and upload your files

    Make sure it has an index.html file

    Hooray! You did this last week!

  2. Go to the Settings tab

  3. Go to Pages in the sidebar

  4. Select the main branch

  5. You are almost done!. Check that Github is preparing your site to be live.

    Go to the Actions tab

    Notice that is says "pages build and deployment. It will turn green like the second one in the image when it is done.

  6. Go back to the Settings -> Pages

    There is now a link to your website that anyone can go to and see.

  7. Now it Github will rebuild your app and make it available to the world every time you make changes and upload your files.
